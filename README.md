# Forex Signal Bot App

**Objetivo del proyecto:**  
Desarrollar una aplicación móvil multiplataforma (iOS y Android) que genere sugerencias de entrada en el mercado Forex usando indicadores técnicos simples (Bandas de Bollinger, RSI, y Medias Móviles).

**Rol del agente de IA (Cursor):**
- Crear la arquitectura base del proyecto.
- Implementar la lógica de análisis técnico.
- Diseñar una interfaz intuitiva con alertas visuales.
- Generar documentación y guías de uso.
